------------------------------------------------------------------------

  - Integrated FieldError component and parser into all 8 frontend forms to display red-colored inline warning labels instead of raw technical strings.

  - Added email VARCHAR(150) column to nguoi_dung_he_thong table to avoid cross-schema queries.

  - Extended GET /api/v1/me to retrieve and return employee email parameter.
* Express Query Validation Refactoring
  - Refactored validateQuery middleware in validate.js to shadow req.query using Object.defineProperty on the request instance, successfully resolving Express 5 getter-only reassignment crashes.
  - Verified GET /api/v1/phong query filters and type coercion.
